# ML/DL 필수 이론 및 작성 코드
---------------------------------

- ### Contents : _Tensorflow 없이 Numpy로 Deep learning 코딩하기_
>  Why?
>1. ML/DL의 수식을 기반으로 직접 코딩이 가능한 수준의 지식보유 <br>
>2. Numpy와 python을 다루는데 능숙함
>3. 즉, 딥러닝을 numpy 기반의 python class로 작성이 가능<br> 
    (**Tensorflow 등 High-level API를 사용하지 않음**)
